\begin{abstract}
Recent work demonstrates that features learned from the visual appearance of a web page can help to improve \ac{LTR} performance. 
In this work, we introduce two feature extraction methods that significantly improve performance of \ac{LTR} with visual features, namely: 
\begin{inparaenum}[(i)]
\item transfer learning from a pre-trained image recognition model, and
\item synthetic generated saliency heatmaps from the web page snapshots.
\end{inparaenum}\ 
To support our work we introduce and release \datasetname, a rich and diverse dataset for \ac{LTR} with visual features.
\datasetname~consists of visual snapshots, non-visual features and relevance judgments for ClueWeb12 documents and TREC Web Track.
The visual snapshots are rendered with both styling and images using the Wayback Machine and ClueWeb12 online rendering service.
Using \datasetname, we confirm that visual features improve \ac{LTR} performance
and show the effectiveness of the newly introduced methods.
\end{abstract}
